SAP CodeJam Lingen

Möchten Sie sich mit anderen Entwicklern vernetzen und gleichzeitig neues und Aufregendes erkunden? Unter diesem Motto startete in Lingen das erste SAP Developer CodeJam* Event am vergangenen Freitag. Wir haben Ingo Bräuninger von der SAP eingeladen uns die neuesten Technologien wie ABAP in Eclipse und HANA näherzubringen. Ingo Bräuninger ist Product Expert im Team Products & Innovation PI Core Platform ABAP Server und arbeitet zudem auch im Developer Team Custom Code Migration to S/4HANA. Weiter ist er Weltweit auf Community Events und CodeJams unterwegs. Bei diesem Event wurde den Teilnehmern gezeigt, wie man unter Zuhilfenahme von Eclipse ABAP Programme entwickeln kann. Weitere Topics des Tages waren das Debugging mit ABAP in Eclipse, die Nutzung von Core Data Services, Troubleshooting einer Fiori App mit ABAP Profiling und Troubleshooting einer Fiori App unter Verwendung von Dynamic Logpoints.

Project History

Weiter wurden den Teilnehmern die Entwicklung der ABAP Platform in den vergangenen Jahren nähergebracht. Auch wurde unter Zuhilfenahme praktischer Beispiele die Effizienz von Eclipse beim Programmieren in ABAP aufgezeigt und auch, wie sich Eclipse in die Architektur der Entwicklungsumgebung integriert. SAP unterstützt, in Form von Plugins, dabei immer die letzten beiden Eclipse Versionen, sagte Ingo Bräuniger als Ihn ein Teilnehmer nach der zurzeit aktuellen Eclipse Version fragte. Auch war er den ganzen Tag über Ansprechpartner für die Teilnehmer und beantwortete die Ihn gestellten Fragen im kompletten Spektrum der SAP Entwicklung. Auf unserer Webseite www.buescher.org finden Sie die komplette Opening Session mit Ingo Bräuninger. Nehmen Sie sich gerne die Zeit und schauen sie sich an.

Zu den Veranstaltungsthemen zählen die folgenden: Jedes Ereignis basiert auf einem einzigen Thema zu mehreren SAP-Technologien. Entscheidend ist, dass für jede Technologie eine Test- oder Entwickleredition verfügbar ist und jeder Teilnehmer nach dem Verlassen der Veranstaltung weiterarbeiten kann. Dies ist auch der Fall Warum umfasst die Liste nicht "alle" Technologien von SAP. Es gibt natürlich verschiedene Haupttechnologien, aber wenn Sie eine zusätzliche Anfrage haben und eine Test- / Entwickler-Edition verfügbar ist, wenden Sie sich bitte an uns!

Insights

SAP CodeJam ist eine 5 bis 6-stündige praktische Kodierungs- und Netzwerk-Veranstaltung, bei der die Teilnehmer ihr Wissen teilen und gemeinsam mit SAP-Technologien, -Plattformen und -Tools in einer unterhaltsamen und zwanglosen Umgebung zusammenarbeiten. Die Veranstaltungen sind auf die Entwicklergemeinde ausgerichtet und werden von SAP unterstützt. Dabei werden die im SAP Developer Center verfügbaren Technologien untersucht.

Innerhalb dieser Technologien haben wir mehrere Fokusbereiche für die CodeJam-Events: SAP HANA XS (available for SPS10) SAP HANA XS (with XSA available in SAP HANA SPS 11) SAPUI5 (available via the Web IDE or through OpenUI5) Java Development (via SAP HANA Cloud Platform using several services such as Gamification, Identity, Translation, etc) Mobile (with SAP HANA Cloud Platform and SAP WebIDE) SAP Jam collaboration environment (SAP Jam | Cloud Collaboration Tools & Software | SAP) SAP Portal (SAP HANA Cloud Platform, Portal Service) ABAP for SAP HANA Fundamentals ABAP for SAP HANA with Fiori Additional Topics SAP Gateway SAP SQL Anywhere SAP Lumira (Introductory look non SDK version) SAP HANA Cloud Integration ABAP in Eclipse (with SDK) SAP IoT
